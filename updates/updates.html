Sunset Engine

C++/OpenGL/GL3W/GLSL

A game engine specifically for making space games, sims, shooters, anything with planetary bodies. A graphics programming/3D modelling/game design project end goal is to develop a space exploration game using the engine.

Current features include, Entity Component System design focused on modularity, Planet generation, Lighting and point light shadows, Large rendering distances for planetary and solar scale. Rigidbody physics simulation, Raycasting, Ellipsoid and Triangle collisions.
